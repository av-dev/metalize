# Sequence

## name

- Type: `string`

## start

- Type: `string`

Start value

## min

- Type: `string`

Minimum value

## max

- Type: `string`

Maximum value

## cycle

- Type: `boolean`

Sequence looping when max value is reached

## increment

- Type: `string`

Determines which number will be added to the current value of the sequence
