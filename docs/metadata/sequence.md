# Sequence

## name

- Type: `string`

## start

- Type: `String`

Start value

## min

- Type: `String`

Minimum value

## max

- Type: `String`

Maximum value

## cycle

- Type: `Boolean`

Sequence looping when max value is reached

## increment

- Type: `String`

Determines which number will be added to the current value of the sequence
