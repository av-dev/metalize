# Configuration

## dialect

- Type: `'postgres' | 'mysql'`
- Default: `null`
- Required: `true`

## connectionConfig

- Type: `object`
- Default: `null`
- Required: `true`

Connection configuration object for [node-postgres](https://node-postgres.com/features/connecting#programmatic) or [node-mysql2](https://github.com/sidorares/node-mysql2#api-and-configuration)
